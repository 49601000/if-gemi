# conversation.html 仕様・マッピング

`conversation.html` は、比喩/構造化テキストを「ジェミ師匠と僕の会話」に変換するための**プロンプト生成ツール**です。  
このページの各UI値は、コード側で分岐して会話を決定するのではなく、主に `buildPrompt()` 内の指示文として埋め込まれ、LLMの出力傾向を調整します。

## 設計スタンス
- 6パラメータは deterministic control ではなく **prompt steering parameter**
- 同じ値でもモデル差・モデルバージョン差で効き方は多少変動する
- 最終品質は実出力を見ながら調整する前提

## UIパラメータ → プロンプト反映先
| UIパラメータ | JS変数 | 反映先ブロック | 役割 |
| --- | --- | --- | --- |
| 題材の呼び名 | `subjectName` | `【今回の題材】` | 今回扱う比喩・構造の名前 |
| 師匠キャラ | `mentorPreset` / `mentorCustom` | `【師匠キャラ】` | ジェミ師匠の人格指定 |
| 弟子キャラ | `studentPreset` / `studentCustom` | `【弟子キャラ】` | 僕側の反応タイプ |
| 会話の用途 | `purpose` | `【用途】` | カクヨム/SNS/教材など用途指定 |
| 文字数目安 | `targetLength` | `【出力制御】` + 文字数指示文 | 出力長の目安 |
| 思想・価値観 | `thoughts` | `【思想・価値観】` | 会話に混ぜる価値観素材 |
| 思想の反映タイミング | `philosophyTiming` | `【思想の反映タイミング】` + `【思想反映ルール】` | 思想を効かせる位置制御 |
| ジェミ師匠のひとこと ON/OFF | `includeJemiHitokoto` | `【ジェミ師匠のひとこと設定】` + `【出力形式】` | ひとこと層の有無 |
| ひとこと個数 | `jemiHitokotoCount` | `【ジェミ師匠のひとこと設定】` | ひとことの目安件数 |
| 構造分解度 | `structureDepth` | `【出力制御】` | 論点分解/対応明示の厚み |
| 会話劇度 | `dialogueDrama` | `【出力制御】` | 会話の芝居感 |
| 弟子の理解遅延度 | `studentDelay` | `【出力制御】` | 僕の理解速度 |
| 師匠の説明圧 | `teacherDensity` | `【出力制御】` | ジェミ師匠の説明密度 |

| パンチライン強度 | `punchlineStrength` | `【出力制御】` | 終盤の刺し具合 |
| 本文入力 | `source` | `【入力本文】` | 比喩・構造化出力・元ネタ本文 |

## 6パラメータの反映方式
`buildPrompt()` では `X/10` だけでなく、`scaleLabels` と `getScaleLabel()` で意味ラベルを併記します。

出力形式:

```text
- 構造分解度：8/10（高め。論点の分解と対応関係を明確にする）
```

`getScaleLabel(id, value)` の段階:
- `0-1`: 最小
- `2-3`: 低め
- `4-6`: 標準
- `7-8`: 高め
- `9-10`: 最大

## 思想・価値観の反映タイミング
`philosophyTiming` は次の4モード:
- `all`: 全体に反映
- `twist`: 終盤のどんでん返しで反映（デフォルト）
- `final`: 締めの一言だけに反映
- `none`: 反映しない

`thoughts` が空、または `none` の場合も、プロンプトに状態を明示して誤適用を防ぎます。

## ジェミ師匠のひとこと
`includeJemiHitokoto` が ON のとき:
- `【師匠と弟子の会話】` の後、`【構造メモ】` の前に挿入
- `jemiHitokotoCount`（1-5、既定3）を件数目安として要求
- 重要ワードを拾い、短文で**教訓・皮肉・補足ツッコミ**のいずれかを軸に生成
- 単なる本文要約の焼き直しを禁止

OFF のとき:
- `【ジェミ師匠のひとこと】は出力しない` を明示

## DSLレイヤ（口調と代名詞ルール）
`mentorSpeechDsl` + `conversationConstraints` をプロンプト前段に注入します。

主な制約:
- 師匠の一人称: 基本 `ワイ`、たまに `ワシ`
- 師匠の二人称: `お前` または `弟子` のみ
- 弟子の一人称: `僕`（不自然なら省略可）
- 弟子が師匠を呼ぶ語: `師匠` 固定
- 禁止代名詞: `あなた` `君` `きみ` `俺` `私` `自分` など
- 長文化・説明臭・概念ラベル化・セミナー調リアクションを抑制

## 出力フォーマット
標準:
1. `【師匠と弟子の会話】`
2. `【ジェミ師匠のひとこと】`（ON時のみ）
3. `【構造メモ】`

会話ラベル例（表示名）:
- `ジェミ師匠「…」`
- `僕「…」`

## 保存仕様（localStorage）
conversationページ固有キー:

| 保存対象 | localStorageキー | 中身 |
| --- | --- | --- |
| 6スライダーの保存済み設定 | `moshiJemi_conversation_ranges` | 6パラメータ値 |
| ページ全体状態 | `moshiJemi_conversation_state` | 入力欄・出力欄・思想設定・ひとこと設定・コピー状態など |

補足:
- 入力/変更時はデバウンス保存（`scheduleSaveConversationState`）
- 生成時は即保存
- 「設定を保存」ボタンは `ranges` を永続保存し、起動時既定値として再利用
