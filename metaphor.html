Metaphor Debugger / 比喩生成MOD

比喩生成を「理論辞書 → 対応表 → 応用例 → 評価基準 → 失敗診断 → 生成フロー」として扱うUIです。既存デザインを崩さず、LLMプロンプト部品として再利用しやすい構造を目指しました。

タブを切り替えて、比喩の理論・対応・応用・評価・失敗・生成フローを順番に確認できます。

比喩プロンプトビルダー

元ネタ本文・比喩の目的・source domain を選ぶと、Gateチェック入りの実用プロンプトを生成します。

比喩理論辞書

理論名・提唱者・核心概念・具体例をカードで表示し、比喩の土台理解を支援します。

source / target 対応表

比喩生成の内部ロジックとして使える、source domain と target domain の関係を整理します。

source domain target domain 種類 メモ

日常表現への応用

抽象理論を生きたパターン集に変換し、LLMが参照しやすい形で提示します。

上手い比喩の定義

4理論を評価基準に変換し、良い比喩の条件を構造化します。

実装者向けメモ

  • 構造対応: source / target の役割が揃っているか。
  • イメージ力: sensory bundle が豊かか。
  • バグ検査: 対応に矛盾がないか。
  • 伝わる力: 読者の文化共有を確認する。

失敗パターン診断

F-1〜F-5の失敗類型を定義し、どこで比喩が壊れるかを診断できるようにします。

比喩生成フロー

生成フローをチェックゲートとともに可視化し、評価・修正の順序を明示します。
